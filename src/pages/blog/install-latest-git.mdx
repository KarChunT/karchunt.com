# Install latest Git

[PPA Git](https://launchpad.net/~git-core/+archive/ubuntu/ppa?ref=itsfoss.com)

1. Install stable Git version

   ```bash
   sudo add-apt-repository ppa:git-core/ppa
   sudo apt-get update
   sudo apt-get install git
   ```

2. Check Git version
   ```bash
   git --version
   ```
