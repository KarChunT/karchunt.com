## Steps of installing Java OpenJDK 17

<Steps>

### Update **apt** package

```bash
sudo apt update
```

### Install OpenJDK and JRE 17

```bash
sudo apt-get install openjdk-17-jdk openjdk-17-jre -y
```

### Check Java Version

```bash
java -version
```

</Steps>
