## Edit hostname on Ubuntu

<Steps>

### Edit `/etc/hostname`

```bash
sudo nano /etc/hostname
```

### Replace any hostname you like. Save the file.

```bash filename="/etc/hostname"
hello-laptop
```

### Reboot the system

```bash
sudo reboot
```

### Check your hostname

```bash
hostname
```

</Steps>
