# Installation

## Snap

```bash
sudo snap install task --classic
```

## Binary

Download the binary from [release page](https://github.com/go-task/task/releases) and add it to your `PATH`.

```bash
wget https://github.com/go-task/task/releases/download/v3.43.3/task_linux_amd64.deb
sudo dpkg -i task_linux_amd64.deb
```

## Script

Override the installation directory with `-b` flag.

```bash
sh -c "$(curl --location https://taskfile.dev/install.sh)" -- -d -b /usr/local/bin
```

## GitHub Action

You can understand more about this [action](https://github.com/arduino/setup-task) on GitHub.

```yaml
- name: Install Task
  uses: arduino/setup-task@v2
  with:
    version: 3.x
    repo-token: ${{ secrets.GITHUB_TOKEN }}
```
