## Steps to edit hostname on Ubuntu

1. Edit `/etc/hostname`

   ```bash
   sudo nano /etc/hostname
   ```

2. Replace any hostname you like. Save the file.

   ```bash filename="/etc/hostname"
   hello-laptop
   ```

3. Reboot the system

   ```bash
   sudo reboot
   ```

4. Check your hostname
   ```bash
   hostname
   ```
