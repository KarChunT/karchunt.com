## Steps of installing Java OpenJDK 17

1. Update **apt** package

   ```bash
   sudo apt update
   ```

2. Install OpenJDK and JRE 17

   ```bash
   sudo apt-get install openjdk-17-jdk openjdk-17-jre -y
   ```

3. Check Java Version
   ```bash
   java -version
   ```
