## Learn how to install Maven MVN

1. Update **apt** package

   ```bash
   sudo apt update
   ```

2. Install Maven

   ```bash
   sudo apt-get install maven -y
   ```

3. Check Maven Version
   ```bash
   mvn -version
   ```
